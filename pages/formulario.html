Algunos atributos de un form

  • name ==> nombre con que se enviara su valor
  • value ==> el valor
  • type ==> tipo de input
  • for ==> va en los label. Lo asocia a un id
  • id ==> sirve como identificador unico del elemento. Lo relaciona con su label
  • required ==> obligatorio
  • placeholder
  • selected (para select)
  • checked (checkbox, radio)
  • maxlength, minlength
  • min, max
  • autocomplete ==> puede ser on u off
  • multiple (email, file) ==> si puede mandar mas de un elemento
  • autofocus
  • step (range) ==> pasos
  • accept (file) ==> image/* , .jpg, .pdf
